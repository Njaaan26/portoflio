About Me

Saya adalah seorang mahasiswa di Universitas Muhammadiyah, dengan program studi Informatika. Saya memiliki dedikasi tinggi dalam belajar bidang desain, web, dan berbagai aspek yang terkait dengan Teknologi Informatika. Saya percaya bahwa desain yang menarik dan pengalaman pengguna yang baik adalah kunci kesuksesan sebuah situs web. Oleh karena itu, saya selalu berusaha untuk meningkatkan keterampilan saya dalam HTML, CSS, JavaScript, dan berbagai kerangka kerja pengembangan web untuk menciptakan situs web yang memukau. Selain web development, saya juga tertarik pada berbagai aspek lain dalam dunia IT. Saya senantiasa mencari peluang untuk belajar teknologi terbaru dan terus memantau tren industri. Saya berharap dapat mengaplikasikan pengetahuan dan keterampilan saya untuk memecahkan berbagai masalah dunia nyata. Jelajahi portofolio saya untuk melihat beberapa proyek yang pernah saya kerjakan. Jika Anda tertarik untuk berkolaborasi atau memiliki pertanyaan, jangan ragu untuk menghubungi saya. Saya selalu terbuka untuk peluang baru dan tantangan dalam dunia Teknologi Informatika.
